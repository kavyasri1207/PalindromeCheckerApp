# PalindromeCheckerApp
A simple Java console application that checks whether a given word or sentence is a palindrome.

A palindrome is a word, phrase, or sequence that reads the same forward and backward.

Examples:

madam

racecar

level

Features

Checks if a word or sentence is a palindrome

Ignores spaces and letter case

Simple and beginner-friendly Java code

Runs in the terminal / command prompt

Technologies Used

Java

Scanner class for user input

String manipulation
